Frequently Asked Questions

Is ArtifactKeep really free?

Yes. ArtifactKeep is free to download and use with no subscriptions. If you find it useful, consider supporting development.

Where is my data stored?

All your data is stored locally on your computer in JSON files.

  • macOS: ~/Library/Application Support/com.artifactkeep.app/
  • Windows: %APPDATA%\com.artifactkeep.app\
  • Linux: ~/.local/share/com.artifactkeep.app/

Can I sync my data across devices?

Not currently. ArtifactKeep is local-first with no cloud sync. You can manually copy your data folder or use export/import workflows.

What image formats are supported for metadata extraction?

ArtifactKeep is optimized for extracting metadata from PNG images generated by ComfyUI and Automatic1111 when metadata is embedded in the file.

Does ArtifactKeep work offline?

Yes. ArtifactKeep works offline for all core library features. Internet is only needed to download the app or visit external links (like release notes).

How do I back up my data?

  • Copy the database folder listed above
  • Export content to JSON, CSV, or Markdown
  • Either method can be used later to restore data
